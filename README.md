# player-search-app

Aplicação web desenvolvida com HTML, CSS e JavaScript puro para exibição e consulta de jogadores de futebol. O projeto permite pesquisar atletas por nome em tempo real e filtrar resultados por posição, utilizando manipulação do DOM, eventos, funções reutilizáveis e renderização dinâmica de conteúdo.

Durante o desenvolvimento foram aplicados conceitos como arrays de objetos, métodos de busca e filtragem, refatoração de código, responsividade, separação de responsabilidades e boas práticas de organização do JavaScript.

Tecnologias utilizadas:

HTML5
CSS3
JavaScript (ES6+)
